# Bienvenido a tienda deportiva



#Objetivo general 
Desarrollar un sistema de información para una tienda deportiva que permita gestionar de manera eficiente los usuarios, productos, ventas, inventario, pedidos y proveedores, con el fin de optimizar los procesos administrativos y operativos del negocio, mejorando la atención al cliente y el control de la información 

##Objetivos especificos 

1.Diseñar un módulo de gestión de usuarios
2.Desarrollar un módulo de gestión de productos deportivos 
3.Desarrollar un módulo de gestión de pedidos a proveedores 
4.Implementar un módulo de gestión de entregas
